命令联合执行

连接词 释义
; 前面的命令执行完以后，继续执行后面的命令
` `
`
& 将任务置于后台执行
&& 前面的语句为假则直接出错，后面的也不执行，前面只能为真

命令连接与逻辑控制

运算符 CMD (Windows 命令提示符) PowerShell Bash (Linux/Unix)
&& 前命令成功时执行下一个 7.0+版本支持（旧版用 -and 或 ; 替代） 前命令成功时执行下一个
` ` 前命令失败时执行下一个
& 同步执行多个命令（无顺序依赖） 不支持（用 ; 或 Start-Job） 前命令后台执行并继续下一个
` ` 管道传递输出到下一个命令 管道传递对象（非纯文本）
; 分隔多个命令 同左 顺序执行多个命令

逻辑运算（条件判断）

运算符 CMD PowerShell Bash
-and 不支持 逻辑与（if ($a -eq 1 -and $b -eq 2)） 不支持（用 && 替代）
-or 不支持 逻辑或（if ($a -eq 1 -or $b -eq 2)） 不支持（用 `
!/-not 仅历史命令展开 逻辑非（if (-not $condition)） 条件取反（if ! [ -f file ]）

1. 历史命令

  • CMD: !str 执行最近以 str 开头的历史命令
  • Bash: !n 执行历史记录中第 n 条命令
  • PowerShell: 不支持 ! 历史展开（用 Invoke-History）

2. 后台执行

  • Bash: command & 后台执行
  • PowerShell: Start-Job -ScriptBlock { command }
  • CMD: 无原生后台执行（需借助 start /B）
